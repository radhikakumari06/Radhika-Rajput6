# Single-Page News Website - Aaj Tak Clone

Hi! This is a responsive single-page news portal project that I built to practice clean grid-based card layouts, structured semantic components, and fast styling integrations. It features a layout inspired by the popular Aaj Tak news channel web interface.

## 🔗 Live Link
You can check out the live running website here:
👉 https://radhikakumari06.github.io/Radhika-Rajput6/

## ✨ Key Features
- *Responsive Navigation Banner:* A fully uniform top navbar featuring main quick-links alongside an integrated breaking news alert ticker strip.
- *Hero Showcase Section:* Highlighted main program grid displaying popular show templates like "Seedhi Baat", "Dangal", and "Vardaat" on a visual TV frame.
- *Card-Based News Grid:* Dynamic and clean layout organizing stories into side-by-side graphical information cards (India News, Tech News, and Sports News).
- *Categorized Sidebar:* Easily view trending tags and topics such as Politics, Business, and Entertainment mapped beautifully alongside the main feed.

## 🛠️ Technologies Used
- *HTML5:* Used for semantic layout design, custom container sections, image placements, and document text structure.
- *Tailwind CSS:* Used for fluid multi-column distributions, custom typography accents, modern layout spacing, and full desktop-to-mobile responsiveness.

## 📂 Project Structure
- index.html - The primary landing page containing the complete dashboard layout, hero showcases, updates grid, and categorized news footer.
